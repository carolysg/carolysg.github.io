Sales report using Power BI

About the project

Dashboard for the sales department, developed using fictional data provided by Hashtag Treinamentos. Excel was used as the data source. The sales report aims to track the following indicators:

  • Total revenue;
  • Monthly revenue;
  • Total number of orders;
  • Monthly orders;
  • Total profit;
  • Profit margin;
  • Monthly returns;
  • Analyses by brands, stores, product categories, and more.

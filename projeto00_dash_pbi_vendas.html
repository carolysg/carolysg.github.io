Relatório de Vendas no Power BI

Sobre o projeto

Dashboard para a área de vendas, desenvolvido com dados fictícios fornecidos pela Hashtag Treinamentos. O excel foi utilizado como fonte de dados. O relatório de vendas tem como objetivo o acompanhamento dos seguintes indicadores:

  • Faturamento total;
  • Faturamento mensal;
  • Total de pedidos;
  • Pedidos mensais;
  • Lucro total;
  • Margem de lucro;
  • Devoluções mensais;
  • Análises por marcas, lojas, categorias de produtos, entre outras.

Sobre o projeto

Este é o projeto final do módulo Técnicas de Programação I do curso da Ada Tech - LM Tech Data Talents.

O projeto consiste na análise de dados acerca de investidores.
São 3 arquivos CSV, sendo um de cadastro e dois de investimentos, conforme mostrado abaixo:

    Cadastro (características dos investidores):
  • id : Identificação do investidor
  • Nome : Nome completo do investidor
  • Sexo : Feminino ou masculino
  • Nasc : Data de nascimento
  • Idade : Idade do investidor
  • Email : E-mail do investidor
  • Lat_long : Dados geográficos do investidor
  • Job : Profissão do investidor ou do seu responsável (em inglês)
  • Estado : Estado onde o investidor reside

    • Investimento (características dos investimentos):
  • id : Identificação do investidor
  • Salario : Renda do investidor ou seu responsável
  • Perfil : Perfil de investimento
  • Ativo01, Ativo02, Ativo03, Ativo04, Ativo05, Ativo06, Ativo07 : O quanto de dinheiro o investidor colocou em cada ativo
  • Nota1, Nota2 : O quanto o investidor está satisfeito com nossa empresa (2 avaliações)

O projeto foi realizado em grupo (3 pessoas) e os membros podem ser observados no repositório do Github.

As bases de investimentos foram concatenadas e as linhas duplicadas foram removidas, resultando em um dataframe com 38754 linhas e 12 colunas:

Depois, o dataframe de cadastro também foi concatenado ao dataframe de investimentos utilizando a chave "id". Dessa forma, as análises se tornam mais fáceis, limitadas a um único dataframe. Antes de iniciar as análises de fato, verificou-se a quantidade de dados faltantes no dataframe.

O tratamento de valores nulos seguiu a seguinte lógica:

  • Para as colunas de ativos (Ativo01 até Ativo07), substituímos os valores faltantes por zero, já que um investimento de R$ 0,00 é a mesma coisa que não colocar nenhum valor no ativo.
  • Para as notas (Nota1 e Nota2), quando pelo menos uma das notas foi dada, adicionamos as notas que foram dadas na outra amostragem pelo mesmo participante.
  • Quando as duas notas eram faltantes, o valor foi preenchido com a média de cada uma das notas (das outras pessoas).

Dessa forma, todos os valores nulos foram preenchidos.

Manipulação de dados e análise exploratória

Para entender melhor o comportamento dos dados, uma descrição das variáveis numéricas foi realizada:

Após todas as alterações realizadas, o dataframe final ficou com a seguinte estrutura:

A partir daqui, algumas manipulações foram feitas com o intuito de explorar os dados e melhorar seu entendimento:

  • Criação da coluna "Total" com a soma do valor de todos os ativos por investidor.
  • Criação da coluna "Média" com a média das notas 1 e 2.
  • Criação da coluna "Domínio" com o domínio do e-mail de cada investidor.
  • Alteração da coluna "Salário" para "Renda", alterando o tipo da coluna para numérica.
  • Alteração dos valores da coluna "Sexo" para "Masculino" e "Feminino".

A coluna "Média" foi então transformada em número inteiro e uma nova coluna "NPS" foi criada. Essa coluna mostra se o investidor é promotor, neutro ou detrator, de acordo com a seguinte regra:

  • Promotor: média das notas maior ou igual a 80.
  • Neutro: média das notas entre 50 e 79.
  • Detrator: média das notas inferior a 50.

Dashboard for a medical clinic

About the project

Dashboard for a medical clinic, created using fictional data provided by Simplifica Treinamentos. The analysis of patient visits was based on the following information:

  • The clinic serves both insurance-covered and private patients;
  • Some doctors may request follow-up appointments for their patients, and if these occur within 90 days, these consultations—termed 'follow-up'—are not charged;
  • Doctors may request exams for patients, which are linked to the company's database.

Thus, the report was generated to analyze the following parameters:

  • Visits by doctor;
  • Visits by specialty;
  • Visits by period;
  • Visits by payment type;
  • Revenue by payment type.
